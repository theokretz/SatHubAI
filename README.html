# QGIS Plugin - SatHubAI This PyQGIS plugin was developed as part of my bachelor's thesis. ## Current Features - Lets the User select an area on the map - Returns a True Color Image of the selected area for a specific time frame using Sentinel Hub. - Downloads the True Color Image in preferred file format. ## Acknowledgements This project uses code from the [Sentinel Hub Python package documentation](https://sentinelhub-py.readthedocs.io/en/latest/examples/process_request.html).
